1. CSS Animation for Interactive Elements

How it extends basic requirements: Adds smooth hover animations to buttons, enhancing user interaction.

Code used:

                button:hover {
                    background-color: #ff6600;
                    transform: scale(1.1);
                    transition: all 0.3s ease-in-out;
                }

2. Responsive Design for Mobile

How it extends basic requirements: Implements a flexible grid layout that adapts to different screen sizes.

Code used:

                @media screen and (max-width: 768px) {
                    .container {
                        display: flex;
                        flex-direction: column;
                    }
                }
